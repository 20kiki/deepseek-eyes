# Contributing

Thanks for your interest in improving deepseek-eyes.

## How to Contribute

### Report Bugs
Open an issue using the [Bug Report](https://github.com/20kiki/deepseek-eyes/issues/new?template=bug_report.md) template.

### Suggest Features
Open an issue using the [Feature Request](https://github.com/20kiki/deepseek-eyes/issues/new?template=feature_request.md) template.

### Submit Changes
1. Fork the repo
2. Create a branch (`git checkout -b fix/some-bug`)
3. Commit with conventional format (`feat: ...`, `fix: ...`)
4. Push and open a PR

## Commit Convention

```
feat: add new feature
fix: fix a bug
docs: update documentation
```
